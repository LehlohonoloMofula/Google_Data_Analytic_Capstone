# Google Data Analytics Capstone

## Overview
A case study is a common way for employers to assess your skills and observe how you would approach a data-related project. For example, you might be asked to clean and analyze a dataset, create a proposal around how to measure the success of a project, or define success metrics for a given project.

This repository contains case study work completed as part of the Google Data Analytics Capstone.

## Track A: Existing Questions and Datasets
Demonstrate job skills for future interviews using a given dataset for analysis.

## Case Studies

### Cyclistic Bike-Share Analysis
Analyze historical bicycle trip data to identify trends. Understanding how casual riders behave differently from riders with paid memberships is important. This analysis helps executives make decisions about marketing programs and strategies to convert casual riders to riders with annual memberships.

Reference: Case Study 1 - How Does a Bike-Share Navigate Speedy Success?

### Bellabeat Product Analysis
Analyze publicly available fitness tracker data to gain insights into how consumers use these products. Identifying trends in this data is key. This analysis helps Bellabeat, a high-tech manufacturer of health-focused products for women, determine future marketing strategy.

Reference: Case Study 2 - How can a wellness company play it smart?
